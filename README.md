## Вот он мой личный сайтик

Если ты программист и хочешь разобраться, то вот тебе:

- В папке `components` лежат UI компоненты
- В папке `firebase` вся бэкэнд логика (добавление постов, удаление, редактирование)
- В папке `lib` функции хелперы для авторизации и функция с шимер-эффектом для загрузки изображений
- В папке `pages` как ни странно страницы моего сайта
- В папке `public` картинки и прочая статичная метаинфа
- В папке `styles` опять же как ни странно стили, глобальные правда

Еще у меня есть админ панель, но пароль вам от неё я не скажу. Но можете код посмотреть, если интересно, тут `./pages/admin`
